Система использует русифицированную модель Saiga Llama 3, которую необходимо собрать локально.

1. Установите [Ollama](https://ollama.com/).
2. Скачайте модель `saiga_llama3_8b.Q4_K_M.gguf` из репозитория [IlyaGusev на HuggingFace](https://huggingface.co/IlyaGusev/saiga_llama3_8b_gguf/tree/main) и поместите файл в корень проекта.
3. Собрать модель: в командной строке в корневой папке проекта
4. ollama create saiga -f Modelfile
5. Установить библиотеки для python из requirements.txt
6. Поместить необходимые для анализа PDF-файлы в директорию docs/
7. Запустить скрипт индексации:
8. python prepare_data.py
9. Для начала работы с системой и диалога с LLM запустить:
10. python main.py
11. После инициализации появится строка ввода. Система будет отвечать на вопросы строго на основе загруженных в docs/ документов, указывая источник и номер страницы.
12. Для запуска полноценного чат-интерфейса с возможностью фильтрации документов выполнить команду:
13. streamlit run app.py
14. Далее следовать указаниям в командной строке
15. Системный компонент Tesseract OCR устанавливается отдельно в ОС Windows.
